Home Web API

Welcome to our Web API! This is a simple API which supports GET requests. For now there is only one endpoint supported. With this endpoint, you can get information about a user by requesting the email OR the username of an account.

The format of the GET requests looks like this:

GET http://program5-env.eba-6cvwcsgc.us-west-1.elasticbeanstalk.com/api/v1/information/{EMAIL | USERNAME}

For example: GET http://program5-env.eba-6cvwcsgc.us-west-1.elasticbeanstalk.com/api/v1/information/dimpsey

The information you get back for a successful 200 response will look something like this: 
        
{
    "status_code": 200,
    "tracked_cryptos": {
        "BTC": "https://www.messari.io/asset/BTC",
        "ETH": "https://www.messari.io/asset/ETH",
        // ... more cryptos
    },
    "user_info": {
        "email": "email@gmail.com",
        "pfp": "https://bucket.s3.us-west-1.amazonaws.com/pfp.jpeg",
        "username": "username"
    }
}
Note that there is also a 404 response for an account that does not exist.

The next step is to add endpoints for programmatically adding Cryptos to your user list without interacting with the UI (a POST request)!
